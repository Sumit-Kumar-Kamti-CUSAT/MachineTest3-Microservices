# ECommerce Microservices - ASP.NET Core Web API

This project is a Microservices-based E-Commerce backend application developed using ASP.NET Core Web API and .NET 8.

## Microservices Included

* AuthService
* InventoryServices
* OrderService
* SharedLibrary
* ApiGateway

## Features

* JWT Authentication & Authorization
* Role-Based Access Control
* RESTful APIs
* Entity Framework Core
* SQL Server Integration
* CRUD Operations
* Swagger API Documentation
* Microservices Architecture

## Technologies Used

* ASP.NET Core Web API
* C#
* .NET 8
* Entity Framework Core
* SQL Server
* Swagger
* JWT Bearer Authentication

## Project Structure

```bash
ECommerceMicroservices
│
├── AuthService
├── InventoryServices
├── OrderService
├── SharedLibrary
└── ApiGateway
```

## Setup Instructions

1. Clone the repository
2. Open solution in Visual Studio
3. Configure SQL Server connection strings
4. Run Entity Framework migrations
5. Start multiple startup projects
6. Run the application
